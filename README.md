# NetModular.Skins.Pithy

[NetModular](https://github.com/iamoldli/NetModular)的第三方前端皮肤组件`pithy`

## 安装

```
npm i -S netmodular-skins-pithy
```

```js
import PithySkin from 'netmodular-skins-pithy'

WebHost.registerSkin(PithySkin)
```
